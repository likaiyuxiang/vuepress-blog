# 到家商品演进

线上化：

供给阶段 -> 交易阶段 -> 履约阶段

商品信息的内容和质量主要由商家来保证。

商品信息主要由**基础信息**和**售卖信息**组成。

商品由基本信息、Sku、库存、图片、店内分类、标签、扩展属性等构成。

类目属性化：

属性 -》 属性模板 -》 类目

上品时需要的属性多，属性容易漏。

建立标品。达尔文体系和SDP模式



平台SPU=类目信息+关键属性

平台标品=类目信息+关键属性+部分交易属性
